# `@mn/prettier-config`

> Мой конфиг [Prettier](https://prettier.io).

## Как использовать

**Устанавливаем**:

```bash
$ npm install -D @mn/prettier-config
```

```bash
$ yarn add -D @mn/prettier-config
```

**Редактируем `package.json`**:

```jsonc
{
  // ...
  "prettier": "@mn/prettier-config"
}
```
